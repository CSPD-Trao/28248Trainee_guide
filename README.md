# 🪴 28248 Trainee Guide

Welcome to the repository for the **28248 Trainee Guide**, an Obsidian-style digital knowledge base for ICT Trainees at Our Lady of the Angels (AGEID: 28248). This project turns standard Markdown notes into a fully interconnected, searchable, and responsive web experience to help trainees manage site-specific infrastructure, devices, and procedures.

---

## ✨ Features

* **Bidirectional Links:** Navigate seamlessly between connected documentation using standard `[[WikiLinks]]`.
* **Network Graph:** Visualize the relationships between different network concepts, hardware, and procedures.
* **Full-Text Search:** Instantly locate specific IP addresses, guides, or troubleshooting steps across the entire knowledge base.
* **Tagging System:** Organize notes fluidly using tags like `#network`, `#hardware`, or `#admin`.
* **Dark/Light Mode:** Automatic theme switching based on system preferences for comfortable reading.
